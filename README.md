# PTA Auto Answer

一个用于 `https://pintia.cn/problem-sets/*` 的 Tampermonkey 用户脚本，支持题面提取、截图补充、多模态模型答题、模拟输入，以及单题自动答题和整套题目的全自动循环。

## 项目特点

- 自动提取 PTA 页面中的题面文字、代码块、表格等内容
- 遇到图片、画布、复杂区域时自动补充区域截图
- 自动识别编辑器输入区域
- 调用兼容 OpenAI Chat Completions 的模型接口生成答案
- 严格模拟输入到页面编辑器，适配常见 `CodeMirror` / `contenteditable` 场景
- 支持自动执行 `提交本题 -> 确认 -> 下一题`
- 支持 `开启全自动答题` 按钮，循环执行直到无法进入下一题

## 安装方法

1. 安装浏览器扩展 `Tampermonkey`
2. 新建一个用户脚本
3. 将 [auto_answer.js](./auto_answer.js) 的完整内容粘贴进去并保存
4. 打开 PTA 题集页面，例如 `https://pintia.cn/problem-sets/*`

## 推荐模型

推荐优先使用阿里百炼的多模态模型。

原因：

- 更适合 PTA 题面里常见的图片、表格、说明文字混合场景
- 本脚本已经内置多模态消息结构，接入更直接
- 支持 OpenAI 兼容接口，配置成本低

推荐配置示例：

- API 地址：`https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions`
- 模型名称：填写你在阿里百炼中实际可用的多模态模型 ID
- API Key：填写你自己的阿里百炼 Key

脚本仓库中已经做了脱敏处理：

- 不包含真实 `API Key`
- 默认模型名是占位值 `your-bailian-model-id`
- 默认 API 地址改为阿里百炼兼容接口地址

## 自动模式用法

1. 打开 PTA 题目页面
2. 在悬浮窗中填写：
   - `API 地址`
   - `API Key`
   - `模型名称`
   - `编程语言`
3. 点击 `保存 API 配置`
4. 点击 `开始自动答题`

单题自动答题流程：

- 自动提取题目区域
- 自动识别输入框
- 调用 AI 生成答案
- 模拟输入到编辑器
- 自动点击 `提交本题`
- 若出现确认框则自动点击 `确认`
- 自动点击 `下一题`

## 全自动模式用法

点击 `开启全自动答题` 后，脚本会循环执行：

- 提取题目
- 识别输入框
- 调用 AI
- 输入答案
- 提交
- 确认
- 下一题

循环会在以下情况下自动停止：

- 当前题执行失败
- 没有找到 `下一题` 按钮
- 点击 `下一题` 后未检测到题目真正切换

再次点击同一个按钮可主动停止。运行中按钮会变成 `停止全自动答题`。

## 手动模式用法

切换到 `手动模式` 后，可以：

- 自动刷新并预览提取到的题面
- 一键复制题面内容
- 粘贴外部模型生成的答案
- 点击 `粘贴内容并输入` 模拟填入编辑器

## 配置说明

- `API Key` 会保存在用户脚本存储中，仅供你本地使用
- 如果平台页面结构更新，可能需要重新自动识别题目区域或输入框
- 全自动模式建议先在少量题目上测试，再长时间连续运行

## 发布与安全

- 请不要把你自己的真实 `API Key`、私有模型 ID 或其他敏感配置提交到公开仓库
- 如果你准备二次分发这个脚本，建议继续保留占位配置，而不是写入真实账号信息
